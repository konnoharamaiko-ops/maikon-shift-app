# UI改善メモ

## 有給管理の残高設定フォーム
- 現在のダイアログ: max-w-md, rounded-2xl, max-h-[90vh] overflow-y-auto
- 必須/任意セクションに分かれている
- 改善点: ステップ形式にして1画面1項目にする、もっと大きなフォント、視覚的なガイド

## シフト表の名前・時間・色
- getShiftColor: 早番(cyan), 中番(lime), 遅番(orange) - 薄い色で見にくい
- セル内テキスト: text-[9px] sm:text-[10px] - 小さい
- ユーザー名ヘッダー: text-[10px] sm:text-xs - 小さい
- WeekTimelineView: バー内テキスト truncate - 切れる

## 改善方針
### 有給残高設定
- ステップウィザード形式（1/2, 2/2）
- 各項目を大きく表示
- ヘルプテキストをより目立つように
- プログレスバー付き

### シフト表
- セルの色をもっと濃く・コントラスト高く
- フォントサイズを全体的に大きく
- 名前を太字で見やすく
- 時間表示をもっと大きく
